## 🛠️ Setup & Execution

Want to mess around with the solutions or test the logic?

### 1. Clone the Lab

```bash
git clone https://github.com/AuditorDevansh/A_to_Z_DSA.git
cd A_to_Z_DSA

```

### 2. Compile & Run

Navigate to a module and fire up the standard GCC compiler:

```bash
cd C++_basic/Pattern_solving
g++ filename.cpp -o app
./app

```
